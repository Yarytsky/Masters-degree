# Osvita — Освітня платформа

Веб-застосунок для шкіл з вбудованим AI-аналізом текстів та автоматичною генерацією тестів.

## Можливості

- **Журнал** — оцінки, відвідуваність, типи робіт
- **Розклад** — календар уроків та дедлайнів
- **AI-детектор** — перевірка робіт учнів на AI-генерацію (8 метрик: burstiness, AI-щільність, особистий голос, структурна уніформність, n-грами, hedge-слова та ін.)
- **Антиплагіат** — порівняння робіт між учнями (Жакар + триграмне перекриття)
- **AI-генератор питань** — створення тестів через Claude API з вбудованою базою як fallback
- **Профілі** — учнів та вчителів з фото, контактами
- **Аналітика** — графіки успішності, відвідуваності, статистика по предметах
- **Адаптивний дизайн** — повна підтримка мобільних

## Запуск

### Варіант 1: Просто відкрити в браузері

Не працюватиме напряму через `file://` через CORS. Потрібен HTTP сервер.

### Варіант 2: Через Node.js (рекомендовано)

```bash
npm install
npm start
```

Відкрити http://localhost:3000

### Варіант 3: Через Python

```bash
cd src
python3 -m http.server 3000
```

### Варіант 4: VS Code Live Server

Встановити розширення "Live Server", правий клік на `src/index.html` → "Open with Live Server"

## Тестові акаунти

- **Учень:** логін `student`, будь-який пароль (Юрій Ярицький, 11-А клас)
- **Вчитель:** логін `Teacher`, будь-який пароль (Іваненко М.В.)

## Технології

- HTML5 + CSS3 + ES6+ JavaScript (без фреймворків)
- Chart.js 4.4 — графіки
- Google Fonts — Fraunces (заголовки), DM Sans (текст)
- Claude API — AI-аналіз та генерація (працює якщо доступний)
- Fallback — локальна база питань та локальний AI-детектор

## Хостинг

Для повноцінної роботи AI потрібен бекенд-проксі.

### Netlify Drop (без AI)

1. Перетягнути папку `src/` на https://app.netlify.com/drop
2. Готово — сайт працює окрім AI-генерації

### Netlify Functions (з AI)

Треба додати функцію проксі для Anthropic API.
